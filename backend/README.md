# Backend — Spring Boot

Esta pasta ficará com a API do quiz (Spring Boot + Java).

## Planejado

- `GET /api/quiz/questoes` — Retorna lista de questões
- `POST /api/quiz/resultado` — Salva resultado do aluno

## Stack planejada

- Java 17+
- Spring Boot 3
- Spring Data JPA
- H2 (banco em memória) ou PostgreSQL
- Maven
